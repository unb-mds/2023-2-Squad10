  <h2>🎯 Conversor de TXT / JSON </h2>
</div> 

• Este é um script Python para extrair dados dos diários oficiais de prefeituras do Piauí e transformar em JSON.

<!-- Título centralizado -->
<div align="center">
  <h2>👩🏾‍💻 Funcionalidades </h2>
</div> 

Este script oferece as seguintes funcionalidades:

• O código encontra nomes de municípios em um texto usando expressões regulares, retornando o primeiro nome encontrado ou um valor padrão.

• O método extrairDados lê arquivos de texto, procura por "LICITAÇÕES" no texto e extrai informações como nome do município, data e número de ocorrências de "licitação". Os dados são escritos em um arquivo JSON chamado "dados.json".

• O método escritaDatabase lê o arquivo "dados.json", adiciona novos dados e o atualiza, evitando a perda de informações antigas.

• O método extraiGeral executa a extração para vários arquivos em um diretório, chamando extrairDados para cada arquivo e atualizando o banco de dados JSON.
