    Aprimorando Projetos e Estudos

    Período: 26/09/2023 a 02/10/2023

    🔰 Descrição

    Esta sprint tem como objetivo aprimorar a organização do quadro Kanban da equipe, iniciar um processo de pesquisa no Diário Oficial para identificar um tema relevante e oportuno para o próximo trabalho, e realizar um estudo inicial para a implementação de raspadores de dados no projeto.

    📒✔️✍ Objetivos

    📑 Organização do Quadro Kanban

    • Avaliar o estado atual do quadro Kanban e identificar áreas que necessitam de melhoria.
    • Revisar e atualizar as listas de tarefas existentes para garantir que estejam refletindo com precisão o status atual de cada tarefa.
    • Garantir que todas as tarefas estejam atribuídas aos membros da equipe responsáveis e que haja clareza quanto a quem é responsável por cada item. Verificar se todas as tarefas têm prazos definidos e se esses prazos são realistas.
    • Revisar as etiquetas e rótulos usados no quadro Kanban para garantir que estejam consistentes e relevantes.
    • Criar uma lista de verificação ou um conjunto de regras de uso do quadro Kanban para garantir a consistência no futuro.
    • Realizar uma reunião de equipe para discutir as mudanças propostas e obter feedback de todos os membros.
    • Implementar as melhorias acordadas e atualizar o quadro Kanban de acordo com as conclusões da reunião.

      • 📖 Pesquisa no Diário Oficial

    • Atribuir a cada membro do grupo a responsabilidade de examinar uma seção específica do Diário Oficial, como legislação, decretos, portarias, ou editais.
    • Realizar uma análise minuciosa das informações disponíveis nas seções designadas, procurando por tópicos de interesse, mudanças regulatórias ou questões relevantes para o campo de estudo.
    • Documentar quaisquer temas ou informações potencialmente úteis encontradas durante a pesquisa.
    • Realizar uma reunião para compartilhar as descobertas e discutir a viabilidade de cada tema identificado.
    • Com base nas informações coletadas e na discussão em grupo, selecionar um tema principal para o próximo trabalho.
    • Começar a elaborar um plano inicial de pesquisa e definir os próximos passos para desenvolver o trabalho.

      • 👨‍💻 Estudo Inicial de Raspagem de Dados

    • Explorar as técnicas de raspagem de dados, incluindo a web scraping, a coleta de dados de APIs e outros métodos relevantes.
    • Identificar e avaliar as principais ferramentas e bibliotecas usadas para a raspagem de dados.
    • Criar guias e tutoriais para ajudar outros membros a iniciar projetos de raspagem de dados de forma eficaz.

      • 📗📘📚 Criterios de Aceitação

      • Todas as tarefas no quadro Kanban estão atualizadas e atribuídas corretamente.
      • Os prazos para todas as tarefas são claros e realistas.
      • As etiquetas e rótulos são consistentes e úteis para a categorização.
      • A equipe concorda com as mudanças propostas no quadro Kanban.
      • Cada membro do grupo deve ter examinado de forma completa e cuidadosa a seção do Diário Oficial atribuída.
      • Deve ser documentado pelo menos um tema relevante encontrado em cada seção examinada.
      • A seleção final do tema no Diário Oficial deve ser baseada em uma discussão em grupo, considerando a relevância, atualidade e disponibilidade de recursos para pesquisa.
      • O plano inicial de pesquisa deve ser desenvolvido com um cronograma claro para as próximas etapas do trabalho.

      ✔️ Conclusão

      • Esta sprint abrange uma série de atividades que visam melhorar a organização, a pesquisa e o conhecimento necessário para a implementação de raspadores de dados em seu projeto.
