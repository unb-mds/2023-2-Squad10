    Sprint 7

    Período: 16/11/2023 a 24/11/2023

    🌌 Descrição 🌌

    Nesta sprint, concentramos nossos esforços na implementação e aprimoramento de ferramentas fundamentais para a leitura e análise de dados a partir de arquivos PDF. Nosso objetivo principal foi fortalecer a capacidade de nossa linguagem de programação preferida ao desenvolver scripts específicos para o download, conversão e análise de documentos provenientes dos diários oficiais das prefeituras do Piauí. Ao integrar bibliotecas como PyMuPDF, Requests e Selenium, alcançamos marcos significativos, proporcionando uma base sólida para futuras iterações e refinamentos. Essa sprint destaca nosso compromisso em fornecer soluções eficazes para as necessidades de extração e transformação de dados em nosso projeto.

    🚀 Objetivos 🚀

    ✈️ Conversor de Download de PDF ✈️

      Este é um script Python para baixar arquivos PDF de diários oficiais de prefeituras do Piauí.

      git clone https://github.com/unb-mds/2023-2-Squad10.git 
       pip install -r requirements.txt 
       pip install requests 
       pip install selenium 
       pip install BeautifulSoup 
       Baixe o ChromeDriver e coloque-o no diretório do projeto.

      👩🏾‍💻 EXECUTE O SCRIPT 👩🏾‍💻 

       python DownloadPDF.py

      🌏 Conversor de PDF / TXT 🌏

        Este é um script Python para transformar arquivos PDF de diários oficiais de prefeituras do Piauí em TXT.

        git clone https://github.com/unb-mds/2023-2-Squad10.git 
         pip install PyMuPDF

        👩🏾‍💻 EXECUTE O SCRIPT 👩🏾‍💻 

         python ConversorPDF TXT.py

        🏝️ Conversor de TXT / JSON 🏝️

          Este é um script Python para transformar arquivos TXT de diários oficiais de prefeituras do Piauí em JSON Para Analise de dados !

          git clone https://github.com/unb-mds/2023-2-Squad10.git 
           pip install PyMuPDF
           pip install request

          👩🏾‍💻 EXECUTE O SCRIPT 👩🏾‍💻

          ✨ Remove Duplicados ✨

            Este é um script Python que irá remover dados JSON que vier duplicados no arquivo gerado para assim a analise de dados ficar mais precisa !
